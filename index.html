Clona el repo

git clone https://github.com/nick/nombre-repo
Esto crea una copia del repo. Los cambios a partir de ahora serán cambios locales. Necesitas estar autorizado para leer del repo, pero todos los repos tienen habilitada la lectura en la versión https, así que esto se puede hacer.
